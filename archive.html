Memory

About

Down below, you will find a small game that I programmed, it's called Memory. I tried my best to pin down and fix every error that I could find - but it's possible that there are still some errors present. With that being said, don't hesitate to submit a pull request to this repository on GitHub.

Rules

In this section, I will quickly cover all the rules that the game has. But, before I do that, it is worthwhile to mention that there are other mechanics besides the actual game, which you can take advantage of. Those features include the Music button, the Reset button, and the Start button. All of those features should be self explanatory. Here is a list with all the rules:

  • Clicking on the Start and Reset button will randomly lay the cards in rows, face down.
  • If you turn over any two cards, and they match, you get to keep them.
  • If they don't match, they will be turned back over again.
  • Try to remember what was on each card and where it was, so you can uncover multiple pairs in a row.
  • In every round, you will have 30 seconds to uncover all pairs.
  • The game is won (over) when all the cards have been matched.
